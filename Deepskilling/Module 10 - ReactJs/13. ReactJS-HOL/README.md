# bloggerapp

A small React lab that demonstrates conditional rendering in several ways while rendering three components: Book Details, Blog Details, and Course Details.

## What it shows

- Conditional rendering with `if/else`
- Conditional rendering with ternary operators
- Conditional rendering with logical `&&`
- Conditional rendering with `switch`
- Conditional rendering with a lookup object
- Rendering multiple components
- List rendering with `map()`
- Stable keys in React lists
- Extracting repeated UI into components

## Run it

```bash
npm install
npm run dev
```

## Build it

```bash
npm run build
```
